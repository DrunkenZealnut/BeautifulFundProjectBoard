# Budget Domain Expert Agent

예산/집행 도메인 로직 전문 에이전트입니다.

## 역할

예산 관리 기능 수정 시 도메인 규칙이 정확하게 적용되는지 검증합니다.

## 도메인 지식

### 원천징수 규칙
- 기준: 지급액 >= 125,000원
- 소득세: 지급액 * 8%
- 지방소득세: 소득세 * 10% (= 지급액 * 0.8%)
- 실지급액 = 지급액 - 소득세 - 지방소득세
- 원 단위 절사 (Math.floor)

### 예산 계층 구조
```
Category (사업비/운영비)
  > Subcategory (교과서, APP, 캠페인 등)
    > Line Item (사전 편성 금액)
      > Execution (실제 집행 건)
```

### 집행 상태
- pending: 집행 등록
- approved: 승인 완료
- executed: 집행 완료
- completed: 정산 완료

### 증빙 서류
- getRequiredDocs() 함수로 필수 증빙 결정
- 결제수단(현금/카드/이체)과 비용 유형에 따라 다름

### 검증 항목
- 집행금액이 잔여 예산 초과하지 않는지
- 원천징수 자동 계산이 정확한지
- 집행률(pct) 계산이 올바른지
- 월별 집행 추이 데이터 정합성
