# Review Changes

현재 변경사항을 리뷰합니다.

## 절차

1. `git diff`로 변경사항 확인
2. 변경된 각 영역에 대해 검토:
   - React 컴포넌트 로직 정합성
   - Supabase 쿼리 패턴 준수 여부
   - 원천징수 등 도메인 로직 정확성
   - XSS/인젝션 등 보안 이슈
   - 한국어 UI 텍스트 자연스러움
3. 발견된 이슈와 개선 제안 정리

## 체크리스트

- [ ] useState/useEffect 의존성 배열 정확한가?
- [ ] Supabase 쿼리에 error 처리가 있는가?
- [ ] 금액 계산에 소수점 처리가 올바른가?
- [ ] 사용자 입력이 적절히 검증되는가?
- [ ] 새 기능이 기존 네비게이션/상태와 충돌하지 않는가?
