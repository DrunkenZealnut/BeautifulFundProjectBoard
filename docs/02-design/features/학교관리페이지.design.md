# Design: 학교관리페이지

> Plan 참조: `docs/01-plan/features/학교관리페이지.plan.md`

## 0. NEIS Open API 연동

### 0.1 API 개요

학교 등록 시 NEIS(나이스) 교육정보 Open API를 통해 학교 기본정보를 자동으로 가져옵니다.

- **API**: `https://open.neis.go.kr/hub/schoolInfo`
- **인증**: API Key (환경변수 `NEIS_API_KEY`)
- **응답**: JSON

### 0.2 NEIS → schools 필드 매핑

| NEIS 필드 | 설명 | → schools 필드 |
|-----------|------|----------------|
| `SCHUL_NM` | 학교명 | `name` |
| `ORG_RDNMA` + `ORG_RDNDA` | 도로명주소 | `address` |
| `ORG_TELNO` | 전화번호 | `phone` |
| `HMPG_ADRES` | 홈페이지 | `website` |
| `ATPT_OFCDC_SC_NM` | 교육청명 | `education_office` (신규 필드) |
| `SD_SCHUL_CODE` | 학교코드 | `neis_code` (신규 필드) |
| `ATPT_OFCDC_SC_CODE` | 교육청코드 | `neis_office_code` (신규 필드) |
| `FOND_SC_NM` | 설립구분 (공립/사립) | `foundation_type` (신규 필드) |
| `COEDU_SC_NM` | 남녀공학 구분 | `coeducation` (신규 필드) |
| `FOND_YMD` | 설립일 | `founded_date` (신규 필드) |
| `ORG_FAXNO` | FAX | `fax` (신규 필드) |

### 0.3 학교 검색 Flow

```text
[학교 등록] 클릭
    ↓
학교명 입력 (예: "인천전자마이스터")
    ↓
[NEIS 검색] 버튼 클릭
    ↓
NEIS API 호출: /hub/schoolInfo?SCHUL_NM=인천전자마이스터
    ↓
검색 결과 목록 표시 (복수 가능)
    ↓
사용자가 선택 → 기본정보 자동 채움
    ↓
담당교사, 전공, 학생수 등 수동 입력
    ↓
[저장]
```

### 0.4 API 호출 (클라이언트 측)

NEIS API는 CORS를 지원하므로 프론트엔드에서 직접 호출 가능:

```javascript
const searchSchoolFromNEIS = async (schoolName) => {
    const NEIS_KEY = 'aab253d940424f76a2de2bb786524694';
    const url = `https://open.neis.go.kr/hub/schoolInfo?KEY=${NEIS_KEY}&Type=json&pIndex=1&pSize=10&SCHUL_NM=${encodeURIComponent(schoolName)}`;
    const res = await fetch(url);
    const json = await res.json();
    if (json.schoolInfo) {
        return json.schoolInfo[1].row.map(r => ({
            name: r.SCHUL_NM,
            address: (r.ORG_RDNMA || '') + ' ' + (r.ORG_RDNDA || ''),
            phone: r.ORG_TELNO || '',
            fax: r.ORG_FAXNO || '',
            website: r.HMPG_ADRES || '',
            education_office: r.ATPT_OFCDC_SC_NM || '',
            neis_code: r.SD_SCHUL_CODE || '',
            neis_office_code: r.ATPT_OFCDC_SC_CODE || '',
            foundation_type: r.FOND_SC_NM || '',
            coeducation: r.COEDU_SC_NM || '',
            founded_date: r.FOND_YMD || '',
        }));
    }
    return [];
};
```

## 2. DB 스키마

### 2.1 schools 테이블

```sql
CREATE TABLE IF NOT EXISTS public.schools (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    -- NEIS API auto-fill fields
    name TEXT NOT NULL,
    short_name TEXT,
    address TEXT,
    phone TEXT,
    fax TEXT,
    website TEXT,
    education_office TEXT,
    neis_code TEXT,
    neis_office_code TEXT,
    foundation_type TEXT,
    coeducation TEXT,
    founded_date TEXT,
    -- Manual input fields
    principal TEXT,
    contact_teacher TEXT,
    contact_teacher_phone TEXT,
    contact_teacher_email TEXT,
    departments TEXT[] DEFAULT '{}',
    student_count INTEGER DEFAULT 0,
    student_count_by_grade JSONB DEFAULT '{}',
    notice_url TEXT,
    memo TEXT,
    status TEXT DEFAULT 'active' CHECK (status IN ('active', 'inactive')),
    partnership_date DATE,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 2.2 school_textbooks 테이블

```sql
CREATE TABLE IF NOT EXISTS public.school_textbooks (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    school_id UUID REFERENCES public.schools(id) ON DELETE CASCADE,
    textbook_name TEXT NOT NULL,
    subject TEXT,
    publisher TEXT,
    year INTEGER,
    is_safety_related BOOLEAN DEFAULT false,
    our_textbook BOOLEAN DEFAULT false,
    distribution_status TEXT DEFAULT 'planned'
        CHECK (distribution_status IN ('planned', 'distributed', 'in_use')),
    quantity INTEGER DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 5.6 NEIS 검색 UI 동작

1. 사용자가 학교명을 입력하고 [NEIS 검색] 클릭
2. API 결과가 드롭다운 리스트로 표시 (최대 10건)
3. 학교 선택 시 기본정보 필드 자동 채움 (name, address, phone, fax, website, education_office 등)
4. 자동 채움된 필드는 수동 수정 가능 (편집 잠금 없음)
5. NEIS에 없는 정보(담당교사, 전공, 학생수 등)는 수동 입력
6. 수정 모드에서는 NEIS 재검색 버튼 표시 (정보 갱신용)
