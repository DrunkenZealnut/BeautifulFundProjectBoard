# Design: 메인페이지(대시보드) 스타일 변경

> Plan 참조: `docs/01-plan/features/메인페이지스타일변경.plan.md`

## 1. 디자인 개요

### 1.1 변경 파일
| 파일 | 변경 내용 |
|------|----------|
| `index.html` (CSS ~38-2174줄) | 대시보드 전용 CSS 클래스 추가/수정 |
| `index.html` (renderDashboard ~4660-5190줄) | JSX 마크업 + 인라인 스타일 개선 |

### 1.2 변경하지 않는 파일
- 다른 페이지 렌더 함수 (renderBudget, renderSchedule 등)
- Supabase 쿼리 로직, calculateStats
- 기존 CSS 클래스 (.card, .btn 등 — 다른 페이지에서도 사용)

## 2. 레이아웃 구조

### 2.1 현재 대시보드 구조
```
섹션 타이틀 (🏠 전체 현황)
├── 알림 배너
├── 예산 소진 분석 (card)
├── 카테고리별 과부족 (card)
├── 통계 카드 4개 (stats-grid)
│   ├── 예산 집행률
│   ├── 예정된 일정
│   ├── 갤러리 사진
│   └── 게시글
├── 성과지표 달성현황 (card, 4-column grid)
├── 월별 집행 추이 차트 (Chart.js canvas)
├── 2-column grid
│   ├── 예정된 일정 목록 (card)
│   └── 최근 게시글 목록 (card)
├── 갤러리 썸네일 (card, 3-column grid)
├── 월별 집행 추이 바차트 (card)
├── 카테고리별 집행 현황 (card)
└── 예산 vs 성과 상관관계 (card)
```

### 2.2 새 대시보드 구조
```
환영 배너 (그라데이션 히어로)
├── 사용자 이름 + 날짜 + 인사말
└── 예산 집행률 대형 게이지

알림 배너 (기존 유지)

핵심 지표 카드 4개 (dash-stats-grid)
├── 예산 집행률 (틸 그라데이션)
├── 예정된 일정 (블루 그라데이션)
├── 갤러리 사진 (앰버 그라데이션)
└── 게시글 (퍼플 그라데이션)

예산 인사이트 섹션 (dash-insight-card)
├── 예산 소진 분석 (개선된 프로그레스 바)
└── 카테고리별 과부족 (기존 유지)

성과지표 (dash-kpi-grid)
├── 교과서기초연구 (좌측 컬러 보더)
├── 캠페인 (좌측 컬러 보더)
├── APP (좌측 컬러 보더)
└── 운영위원회 (좌측 컬러 보더)

2-column grid (dash-content-grid)
├── 예정된 일정 (타임라인 스타일)
└── 최근 게시글 (컴팩트 리스트)

차트 & 분석 (기존 유지, 미세 스타일만 조정)
├── 월별 집행 추이
├── 카테고리별 현황
└── 예산 vs 성과
```

## 7. 변경하지 않는 요소

- Chart.js 캔버스 차트 (renderMonthlyChart) — 스타일만 카드 래핑
- 분기별 보고서 버튼 기능
- 알림 배너 기능 (dismissAlert)
- 성과지표 수정 기능 (perfEditing)
- 모든 클릭 이벤트 핸들러
- 갤러리 썸네일 DashboardGalleryThumb 컴포넌트 로직

## 8. 디자인 제약 조건

- **외부 라이브러리 추가 금지** — CSS만으로 구현
- **기존 CSS 클래스 삭제 금지** — `.stat-card`, `.stats-grid` 등은 다른 페이지에서 사용 가능
- **`dash-` 접두어** — 대시보드 전용 스타일은 반드시 `dash-` 접두어 사용
- **인라인 스타일 최소화** — 가능한 CSS 클래스로 이동 (신규 부분만)
