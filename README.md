# 아름다운재단 공익단체 인큐베이팅 지원사업 관리시스템

## 📋 프로젝트 개요

청년노동자인권센터의 **2026 공익단체 인큐베이팅 지원사업**을 위한 종합 사업관리 웹 애플리케이션입니다.

총 사업비 **7천만원**에 대한 예산 집행 관리부터 일정 관리, 갤러리, 게시판, 참여자 관리까지 사업 운영에 필요한 모든 기능을 통합 제공합니다.

## 🚀 주요 기능

### 💰 예산 관리 시스템
- **실시간 예산 집행률 추적** - 총 예산 7천만원 대비 집행 현황
- **원천징수 자동 계산** - 125,000원 초과 시 소득세 8%, 지방소득세 10% 자동 계산
- **증빙서류 체크리스트** - 계정항목별 맞춤형 서류 요구사항 자동 생성
- **아름다운재단 회계 규정 준수** - 수행가이드 기반 모든 회계 규칙 반영

### 📅 사업 일정 관리
- **교육, 캠페인, 회의** 등 카테고리별 일정 분류
- **참여자 현황 추적** - 정원 대비 신청자/출석자 현황
- **담당자 지정** 및 진행 상태 관리
- **달력 뷰 지원**

### 📸 갤러리 시스템
- **현장사진, 결과물, 교육자료** 등 카테고리별 분류
- **태그 시스템** - 검색 및 분류 최적화
- **메타데이터 관리** - 촬영일, 장소, 촬영자 등 상세 정보

### 📢 게시판 시스템
- **공지사항** - 사업 공지 및 안내
- **자료실** - 교육자료, 보고서 등 파일 공유
- **FAQ** - 자주 묻는 질문 및 답변
- **첨부파일 지원** - PDF, DOC, HWP 등 다양한 파일 형식

### 👥 참여자 관리 시스템
- **참여자 등록 및 관리** - 개인정보, 소속기관 등 체계적 관리
- **실시간 출석체크** - QR코드 기반 또는 수동 체크
- **만족도 조사** - 5점 척도 평가 및 피드백 수집
- **Excel 내보내기** - 참석자 명단 및 통계 다운로드

### 📊 통계 및 리포팅
- **실시간 대시보드** - 예산 집행률, 일정 현황, 참여자 통계
- **진행률 시각화** - 프로그레스바 및 차트로 직관적 현황 파악
- **월별/분기별 리포트** 생성 지원

## 🛠 기술 스택

### Frontend
- **React** - 모던 컴포넌트 기반 UI 프레임워크
- **CSS-in-JS** - 스타일 컴포넌트 방식으로 유지보수성 향상
- **Responsive Design** - 모바일/태블릿 완벽 대응

### Backend (권장)
- **Node.js + Express** 또는 **PHP**
- **MySQL/MariaDB** - 관계형 데이터베이스
- **JWT 인증** - 보안 토큰 기반 사용자 인증

### 보안 및 규정 준수
- **HTTPS 필수** - 개인정보보호법 준수
- **데이터 암호화** - 민감정보 암호화 저장
- **백업 정책** - 일일 자동 백업

## 🗄 데이터베이스 구조

### 핵심 테이블
- **users** - 사용자 계정 및 권한 관리
- **budget_executions** - 예산 집행 내역
- **documents** - 증빙서류 관리
- **schedules** - 일정 및 행사 관리
- **participants** - 참여자 등록 및 출석
- **galleries** - 이미지 및 미디어 파일
- **boards** - 게시판 및 공지사항

### 관계형 설계
- 일정(schedules) ↔ 참여자(participants) : 1:N
- 예산집행(budget_executions) ↔ 증빙서류(documents) : 1:N
- 모든 테이블에 작성자, 수정자, 일시 추적

## 🚀 설치 및 실행

### 1. 데이터베이스 설정
```sql
-- MySQL/MariaDB에서 실행
CREATE DATABASE beautiful_fund_project;
USE beautiful_fund_project;
SOURCE database-schema.sql;
```

### 2. 환경 설정
```env
# .env 파일 생성
DB_HOST=localhost
DB_NAME=beautiful_fund_project
DB_USER=your_username
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret
UPLOAD_PATH=./uploads
```

### 3. 애플리케이션 실행
```bash
# React 개발 서버 (예시)
npm install
npm start

# 또는 정적 파일로 빌드
npm run build
```
